알러지 트렌드 분석 가이드

알러젠별 연구 동향, 뉴스, 키워드 트렌드를 한눈에 파악합니다

1. 서비스 소개

알러지 트렌드 분석은 AllergyInsight에 축적된 논문 데이터와 최신 뉴스를 분석하여, 알러젠별 연구 동향, 키워드 트렌드, 논문 컬렉션, 최신 뉴스 등을 시각화된 대시보드로 제공하는 서비스입니다. 알러지 분야의 전체적인 흐름과 개별 알러젠의 심층 분석 정보를 확인할 수 있습니다.

대상 사용자

알러지 연구자 의료 전문가 보건 정책 담당자 알러지 환자 및 보호자

2. 주요 기능

📊

분석 대시보드

전체 알러젠의 연구 트렌드, 논문 수, 주요 키워드를 종합 대시보드에서 한눈에 확인합니다.

🔬

알러젠 분석

개별 알러젠의 연구 동향, 관련 논문 수 변화, 주요 연구 주제를 심층 분석합니다.

📈

키워드 트렌드

알러지 연구 분야의 핵심 키워드 빈도 변화와 신규 등장 키워드를 추적합니다.

📰

최신 뉴스

알러지 관련 최신 뉴스를 자동 수집하여 최근 7일간의 주요 소식을 제공합니다.

📚

논문 컬렉션

주제별, 알러젠별로 분류된 논문 컬렉션을 탐색하고 관심 논문을 확인할 수 있습니다.

💡

인사이트 상세

각 알러젠별 AI 기반 심층 인사이트 리포트를 확인하고 연구 방향을 파악합니다.

3. 사용 방법

  1. 트렌드 분석 페이지 접속
    메인 화면 또는 메뉴에서 "트렌드 분석"을 클릭하여 분석 대시보드로 이동합니다.
  2. 개요 탭에서 전체 현황 파악
    개요 탭에서 전체 알러젠의 연구 트렌드 요약, 최근 논문 수, 주요 키워드 등을 확인합니다.
  3. 알러젠 탐색으로 심층 분석
    관심 있는 알러젠을 선택하면 해당 알러젠의 연구 동향, 관련 논문, 키워드 변화 등 상세 분석 정보를 볼 수 있습니다.
  4. 키워드 트렌드 확인
    키워드 트렌드 메뉴에서 시간에 따른 연구 키워드 변화를 차트로 확인하고, 새로운 연구 흐름을 파악합니다.
  5. 최신 뉴스 확인
    뉴스 탭에서 최근 7일간의 알러지 관련 뉴스를 확인하고 관심 기사를 클릭하여 상세 내용을 읽습니다.

4. 주의사항

데이터 갱신 주기

논문 데이터는 정기적으로 PubMed, Semantic Scholar에서 수집되며, 뉴스는 매일 자동 업데이트됩니다. 최신 데이터 반영까지 일정 시간이 소요될 수 있습니다.

분석 범위

트렌드 분석은 AllergyInsight에 수집된 논문 데이터를 기반으로 하며, 전체 의학 논문을 포괄하지는 않습니다. 연구 참고 자료로 활용하시기 바랍니다.

브라우저 호환성

차트 및 시각화 기능은 최신 브라우저(Chrome, Edge, Safari, Firefox)에서 최적화되어 있습니다. Internet Explorer는 지원하지 않습니다.

5. 자주 묻는 질문

Q. 트렌드 분석을 이용하려면 로그인이 필요한가요?
아니요, 트렌드 분석은 공개 페이지로 로그인 없이 누구나 이용할 수 있습니다.
Q. 논문 데이터는 얼마나 자주 업데이트되나요?
논문 데이터는 PubMed, Semantic Scholar에서 정기적으로 수집됩니다. 뉴스 데이터는 매일 자동으로 갱신됩니다.
Q. 키워드 트렌드 차트는 어떤 기간을 보여주나요?
수집된 논문의 전체 기간에 대한 키워드 빈도 변화를 보여줍니다. 특정 기간을 선택하여 필터링할 수도 있습니다.
Q. 특정 알러젠의 상세 정보는 어떻게 확인하나요?
개요 탭에서 관심 알러젠을 클릭하면 자동으로 알러젠 탐색 탭으로 이동하여 해당 알러젠의 상세 분석 정보(연구 동향, 관련 논문, 키워드 등)를 확인할 수 있습니다.
Q. 분석 결과를 다운로드할 수 있나요?
현재는 화면에서 직접 확인하는 방식만 지원합니다. 향후 PDF 리포트 다운로드 기능이 추가될 예정입니다.
Q. 뉴스는 어디서 수집되나요?
알러지 관련 주요 뉴스 소스에서 RSS 피드를 통해 자동 수집되며, 최근 7일간의 뉴스가 표시됩니다.
Q. 인사이트 상세 리포트는 어떻게 생성되나요?
AI가 각 알러젠에 대한 수집된 논문 데이터를 종합 분석하여 연구 동향, 주요 발견, 향후 전망 등을 포함한 인사이트 리포트를 자동 생성합니다.
