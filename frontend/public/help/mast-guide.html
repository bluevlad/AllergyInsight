MAST 검사 등급 매칭 가이드

병원 · 진단소에서 받은 알러지 검사 결과를 입력하면 식이 · 증상 · 응급 정보를 확인할 수 있습니다

1. 서비스 소개

MAST(Multiple Allergen Simultaneous Test) 는 한 번의 검사로 여러 알러젠에 대한 IgE 반응을 동시에 측정하는 다중 알러지 검사입니다. 본 페이지에서는 병원 또는 진단소에서 받은 MAST 등급(Class 0~4)을 입력하면, 해당 알러젠에 대해 논문 · 전문기관 출처 기반의 식이 관리 · 예상 증상 · 교차반응 · 응급 대처 정보를 매칭해서 보여줍니다.

📌 본 서비스의 위치

본 서비스는 의료 진단 도구가 아니라 정보 매칭 서비스입니다. 이미 의료기관에서 받은 검사 결과를 바탕으로 일상에서 참고할 수 있는 정보를 제공할 뿐, 새로운 진단을 내리지 않습니다.

2. MAST 등급 체계 (Class 0~4)

본 서비스가 사용하는 등급 체계는 다음과 같습니다. 한국 임상에서 통용되는 라벨과 표준 클래스 명칭을 함께 표기합니다.

등급한국어 라벨표준 명칭설명
0음성Class 0 (Negative)알러지 반응 없음 — 제한 없이 섭취 가능
1약양성Class 1 (Weak)약한 반응 — 섭취 후 증상 관찰 권장
2중등도 양성Class 2 (Moderate)중등도 반응 — 섭취 시 주의 관찰
3강양성Class 3 (Strong)강한 반응 — 섭취 제한 권고
4매우 강양성Class 4 (Very Strong)매우 강한 반응, 아나필락시스 위험 — 회피 + 응급약 휴대 권고
⚠️ 다른 검사 결과를 가지고 계신 경우

ImmunoCAP(0~6) 등 다른 검사 체계를 사용하셨다면 등급이 다를 수 있습니다. 검사 종류별 등급은 직접 비교가 어려우므로, 본 페이지에서는 MAST 기준 라벨을 입력해 주세요.

3. 사용 방법

  1. 알러젠 선택 — 한글명, 영문명, 또는 코드로 검색하거나 카테고리 필터로 알러젠을 찾습니다. 현재 본 서비스는 처방 데이터가 풍부하게 준비된 36종(식품 21 + 흡입 15)을 매칭 대상으로 합니다.
  2. MAST 등급 선택 — Class 0(음성)부터 Class 4(매우 강양성)까지 5단계 중 병원/진단소 결과지에 표기된 등급을 선택합니다.
  3. "정보 매칭하기" 클릭 — 입력값을 보내면 즉시 결과 화면으로 이동합니다. 별도 회원가입은 필요 없으며, 입력 정보는 저장되지 않습니다(익명).

4. 결과 화면 구성

입력한 알러젠과 등급에 따라 다음 정보가 자동으로 매칭되어 표시됩니다.

5. 응급 상황 안내

🚨 다음 증상 시 즉시 119

숨가쁨/호흡곤란, 입술/혀/목 부종, 의식 저하, 어지러움, 전신 두드러기 + 호흡 증상, 구토와 함께 발생하는 알러지 반응, 혈압 저하 등 아나필락시스 의심 증상이 나타나면 즉시 119에 전화하시고, 에피네프린 자가주사기(에피펜)가 있다면 즉시 사용하세요.

Class 3(강양성) 이상의 결과를 입력하면 결과 화면 상단에 응급 정보가 자동으로 노출됩니다. 이는 임상적으로 중증 반응 위험이 높은 등급에 대해 항상 응급 절차를 먼저 안내하기 위함입니다.

6. 자주 묻는 질문

Q. 회원가입 없이 사용할 수 있나요?
네. 본 서비스는 비회원 익명 사용을 기본으로 합니다. 입력한 알러젠 · 등급은 서버에 저장되지 않습니다.
Q. 왜 36종만 조회 가능한가요?
의학적으로 검증된 식이/증상/응급 정보가 충분히 채워진 36종에 한정해서 노출하고 있습니다. 출처 부재 또는 정보 부족 상태로 노출하면 사용자에게 잘못된 안내가 될 수 있어, 풍부한 데이터 보유 알러젠만 우선 활성화하는 정책입니다.
Q. 본 서비스 결과로 자가 진단이 가능한가요?
아니요. 본 서비스는 의료 진단을 대체하지 않습니다. 모든 정보는 교육 · 정보 매칭 목적이며, 실제 식이 결정 · 처방 · 응급 대처는 반드시 의료진과 상담하세요.
Q. 검사 결과지에 등급이 0~6으로 표기되어 있어요
CAP/ImmunoCAP 등 다른 검사 체계는 0~6 단계입니다. 본 서비스는 MAST 0~4 기준으로 동작하므로, 검사기관에 결과지의 등급 체계를 확인 후 가장 가까운 MAST 클래스를 선택하시는 것을 권장합니다 (대략 5/6은 Class 4에 가까움).
Q. 약물 정보 / 약 추천을 받을 수 있나요?
현재는 등급별 응급약 휴대 권고 등 일반 안내만 제공합니다. 특정 약품 추천 · 처방 안내는 의약품 광고법 등 법적 제약으로 제공하지 않습니다. 약물 성분 정보는 향후 별도 페이지에서 출처와 함께 제공할 예정입니다.
Q. 사진 업로드로 알러지를 진단해 주세요
현재 단계에서는 사용자 사진을 통한 진단/매칭은 제공하지 않습니다. 추후 논문 출처가 명확한 임상 사진 갤러리 형태로 정보 제공이 검토될 수 있으나, 사용자 사진 분석은 의료기기 인허가 영역이라 별도 절차가 필요합니다.

7. 면책 / 정보 출처 한계

본 서비스의 한계

본 페이지의 모든 정보는 논문 · 전문기관 자료를 큐레이션한 교육 · 정보 매칭 목적이며, 의료 진단 · 처방 · 치료를 대체하지 않습니다. 정확한 진단과 처방은 반드시 의료진과 상담하세요. 자세한 내용은 의료 정보 면책 안내를 참고하세요.
