# 사용자 가이드 (User Guide)

## 시작하기

### 접속 URL

| 서비스 | URL | 대상 |
|--------|-----|------|
| Professional | `/pro` | 의료진 |
| Consumer | `/app` | 환자 |
| Admin | `/admin` | 플랫폼 관리자 |
| Analytics | `/analytics` | 모두 (공개) |
| AI 상담 | `/ai/consult` | 모두 (공개) |
| AI 인사이트 | `/ai/insight` | 모두 (공개) |
| 뉴스레터 구독 | `/subscribe` | 모두 (공개) |
| 알러지 리포트 | `/report` | 모두 (공개) |

### 로그인 방법

=== "간편 인증 (PIN)"

    ```
    1. 이름 입력
    2. 전화번호 입력
    3. 접속 PIN 입력 (6자리)
    4. 로그인 버튼 클릭
    ```

=== "Google OAuth"

    ```
    1. "Google로 로그인" 버튼 클릭
    2. Google 계정 선택
    3. 권한 승인
    4. 자동 로그인 완료
    ```

=== "이메일 로그인"

    ```
    1. 이메일 주소 입력
    2. 비밀번호 입력
    3. 로그인 버튼 클릭
    ```

=== "관리자 로그인"

    ```
    1. /admin/login 접속
    2. 관리자 이름 입력
    3. 접속 PIN 입력
    4. 로그인 버튼 클릭
    ```

## 서비스별 가이드

=== "Professional (의료진)"

    ### 대시보드 (`/pro/dashboard`)

    로그인 후 첫 화면입니다.

    ```
    ┌─────────────────────────────────────────────────────────────┐
    │  AllergyInsight Professional                       [로그아웃]│
    ├─────────────────────────────────────────────────────────────┤
    │                                                             │
    │  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐         │
    │  │  총 진단 수  │  │  이번 달    │  │  총 환자 수  │         │
    │  │    152      │  │    28       │  │    89       │         │
    │  └─────────────┘  └─────────────┘  └─────────────┘         │
    │                                                             │
    │  [알러젠 분포 차트]                                          │
    │  [최근 진단 목록]                                            │
    └─────────────────────────────────────────────────────────────┘
    ```

    ### 진단 입력 (`/pro/diagnosis`)

    1. **환자 선택**: 기존 환자 검색 또는 신규 환자 등록
    2. **알러젠 등급 입력**: 16종 알러젠에 대해 0-6 등급 입력
    3. **처방 권고 확인**: 자동 생성된 처방 권고 확인 및 저장

    ### 임상 보고서 (`/pro/clinical-report`)

    - SOAP Note 형식의 근거 기반 보고서 생성
    - GRADE 근거 수준 표시 (⊕⊕⊕⊕ ~ ⊕◯◯◯)
    - 교차반응 분석 및 임상 진술문
    - ICD-10 코드 자동 매핑
    - 참고문헌 목록 (EAACI, AAAAI, WAO 가이드라인)

    ### 환자 관리 (`/pro/patients`)

    - 환자 목록 조회 및 검색
    - 신규 환자 등록 (`/pro/patients/new`)
    - 환자 상세 정보 및 진단 이력

    ### 논문 검색 / Q&A (`/pro/search`, `/pro/qa`)

    - PubMed, Semantic Scholar, Europe PMC, OpenAlex 통합 검색
    - RAG 기반 질의응답 (논문 출처 제공)

=== "Consumer (환자)"

    ### 내 진단 결과 (`/app/my-diagnosis`)

    로그인 후 첫 화면입니다.

    ```
    ┌─────────────────────────────────────────────────────────────┐
    │  AllergyInsight                                    [로그아웃]│
    ├─────────────────────────────────────────────────────────────┤
    │                                                             │
    │  안녕하세요, 홍길동님                                        │
    │                                                             │
    │  최신 진단 결과 (2024-01-15)                                │
    │  🔴 고위험: 땅콩 (Grade 4)                                  │
    │  🟡 주의: 우유 (Grade 2), 집먼지진드기 (Grade 3)             │
    │  🟢 정상: 계란, 밀, 대두...                                 │
    │                                                             │
    │  빠른 메뉴                                                   │
    │  [식품 가이드]  [응급 대처]  [생활 관리]                      │
    └─────────────────────────────────────────────────────────────┘
    ```

    ### 식품 가이드 (`/app/food-guide`)

    - 알러젠별 회피 식품 목록
    - 대체 식품 추천
    - 교차반응 주의 식품
    - 숨겨진 알러젠 경고

    ### 응급 대처 (`/app/emergency`)

    - 증상 심각도별 대처법 (경미/심각)
    - 에피펜 사용법 (단계별 안내)
    - 응급 연락처 (119, 1339)

    ### 생활 관리 (`/app/lifestyle`)

    - 알러젠별 일상 관리 팁
    - 계절별, 상황별 가이드
    - 실내 환경 관리법

    ### 키트 등록 (`/app/kit-register`)

    - 시리얼 번호 + PIN 입력으로 검사 결과 연결

=== "Admin (관리자)"

    ### 접근 방법

    1. `/admin/login` 접속
    2. super_admin 역할 계정으로 로그인

    ### 대시보드 (`/admin/dashboard`)

    ```
    ┌─────────────────────────────────────────────────────────────┐
    │  AllergyInsight Admin                              [로그아웃]│
    ├─────────────────────────────────────────────────────────────┤
    │                                                             │
    │  플랫폼 현황                                                 │
    │  ┌──────────┐  ┌──────────┐  ┌──────────┐  ┌──────────┐    │
    │  │ 총 사용자 │  │ 총 진단  │  │ 총 논문  │  │ 구독자   │    │
    │  │   245    │  │   512   │  │  3,240  │  │   128   │    │
    │  └──────────┘  └──────────┘  └──────────┘  └──────────┘    │
    │                                                             │
    │  사이드바 메뉴:                                               │
    │  • 대시보드                                                  │
    │  • 사용자 관리                                               │
    │  • 알러젠 관리                                               │
    │  • 논문 관리                                                 │
    │  • 조직 관리                                                 │
    │  • 뉴스 관리                                                 │
    │  • 구독자 관리                                               │
    │  • 분석 관리                                                 │
    └─────────────────────────────────────────────────────────────┘
    ```

    ### 주요 관리 기능

    | 메뉴 | 기능 |
    |------|------|
    | **사용자 관리** | 목록 조회, 역할 변경, 활성/비활성, 통계 |
    | **알러젠 관리** | 마스터 데이터 생성/수정/삭제/복원 |
    | **논문 관리** | 수집된 논문 목록, 알러젠 링크 확인 |
    | **조직 관리** | 병원/클리닉 승인/거절 |
    | **뉴스 관리** | 크롤링된 뉴스 관리, 수동 가져오기 |
    | **구독자 관리** | 뉴스레터 구독자 목록, 통계, 수정/삭제 |
    | **분석 관리** | 월간 집계 실행, 키워드/치료법/역학 추출 |

=== "Analytics (공개)"

    ### 접근 방법

    - `/analytics` 접속 (로그인 불필요)

    ### 알러젠 분석 (`/analytics/allergen-analysis`)

    - 알러젠별 논문 언급 추세 차트
    - 상승/하락 알러젠 랭킹
    - 기간별 필터링

    ### 논문 수집 현황 (`/analytics/paper-collection`)

    - 출처별 수집 통계 (PubMed, Semantic Scholar 등)
    - 수집 시계열 차트
    - 최근 수집 현황

    ### 알러젠 뉴스 (`/analytics/allergen-news`)

    - 뉴스 기반 알러젠 동향
    - 최근 뉴스 목록
    - 알러젠별 뉴스 빈도

    ### 종합 트렌드 (`/analytics/comprehensive-trend`)

    - 알러젠 선택 → 종합 분석 뷰
    - 논문 트렌드 + 뉴스 트렌드 + 치료법 + 역학 데이터
    - 차트 + 테이블 결합 뷰

=== "AI Portal (공개)"

    ```
    ┌─────────────────────────────────────────────────────────────┐
    │  AI 알러지 상담                                              │
    ├─────────────────────────────────────────────────────────────┤
    │                                                             │
    │  💬 질문을 입력하세요:                                       │
    │  ┌──────────────────────────────────────────────────────┐  │
    │  │ 땅콩 알러지가 있으면 어떤 식품을 피해야 하나요?        │  │
    │  └──────────────────────────────────────────────────────┘  │
    │  [질문하기]                                                  │
    │                                                             │
    │  🤖 답변:                                                    │
    │  땅콩 알러지 환자는 다음 식품을 피해야 합니다...              │
    │                                                             │
    │  📚 출처:                                                    │
    │  • Smith et al. (2024) - Peanut allergy management          │
    │  • EAACI Guidelines (2023) - Food allergy                   │
    │                                                             │
    │  추천 질문:                                                  │
    │  • 땅콩 알러지의 교차반응은?                                  │
    │  • 아이가 땅콩 알러지인데 학교 급식은?                        │
    └─────────────────────────────────────────────────────────────┘
    ```

    - RAG 기반 답변 (논문 출처 제공)
    - 알러젠별 사전 정의 질문 제공
    - Rate Limit: 10회/분

    ### AI 인사이트 (`/ai/insight`)

    - 알러젠별 종합 분석 보고서
    - 최신 알러지 뉴스 요약
    - 트렌드 시각화

## 뉴스레터 구독

### 구독 방법 (`/subscribe`)

```
1. 이메일 주소 입력
2. 관심 키워드/알러젠 선택 (선택사항)
3. "구독하기" 클릭
4. 이메일로 전송된 인증 코드 입력
5. 구독 완료
```

### 구독 관리 (`/subscribe/manage`)

- 관심 키워드 변경
- 구독 상태 확인

### 구독 해지 (`/subscribe/unsubscribe`)

- 이메일 입력으로 즉시 해지

## 자주 묻는 질문 (FAQ)

=== "일반"

    **Q: 비밀번호를 잊어버렸어요.**

    !!! tip "답변"
        로그인 화면에서 Google OAuth 또는 이메일 재설정을 이용하세요.

    **Q: 다른 기기에서도 접속할 수 있나요?**

    !!! tip "답변"
        네, 동일한 계정으로 여러 기기에서 접속 가능합니다.

=== "Professional"

    **Q: 진단 결과를 수정할 수 있나요?**

    !!! tip "답변"
        환자 상세 페이지에서 해당 진단을 선택하여 수정할 수 있습니다.

    **Q: 임상 보고서의 근거 수준은 어떻게 결정되나요?**

    !!! tip "답변"
        GRADE 체계에 따라 논문의 근거 수준을 A(높음)~D(매우 낮음)으로 평가합니다.

=== "Consumer"

    **Q: 이전 진단 기록을 볼 수 있나요?**

    !!! tip "답변"
        내 진단 페이지에서 모든 과거 진단 기록을 확인할 수 있습니다.

    **Q: AI 상담의 답변을 신뢰해도 되나요?**

    !!! warning "답변"
        AI 상담은 논문 기반 참고 정보를 제공합니다. 의료 결정은 반드시 전문의와 상담하세요.

=== "뉴스레터"

    **Q: 뉴스레터는 얼마나 자주 발송되나요?**

    !!! tip "답변"
        기본적으로 매일 아침 발송됩니다. (설정에 따라 변경 가능)

    **Q: 관심 키워드를 변경하고 싶어요.**

    !!! tip "답변"
        `/subscribe/manage` 페이지에서 변경할 수 있습니다.

## 문제 해결

### 로그인 오류

| 증상 | 해결 방법 |
|------|----------|
| "인증 실패" | PIN 번호 확인 |
| "세션 만료" | 다시 로그인 (JWT 24시간 만료) |
| Google 로그인 실패 | 브라우저 팝업 허용 확인 |
| Admin 로그인 불가 | super_admin 역할 확인 |

### 페이지 오류

| 증상 | 해결 방법 |
|------|----------|
| 페이지 로딩 안됨 | 새로고침 (Ctrl+F5) |
| API 에러 표시 | 네트워크 연결 확인, 서버 상태 확인 |
| 차트 표시 안됨 | 데이터 수집 여부 확인 (Analytics) |

### 지원 요청

!!! note "문제가 지속되면 다음 정보와 함께 문의하세요"
    - 발생 시간
    - 오류 메시지 (스크린샷)
    - 사용 브라우저 / 기기
    - 재현 방법
