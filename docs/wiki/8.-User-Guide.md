# 8. 사용자 가이드 (User Guide)

## 8.1 시작하기

### 접속 URL

| 서비스 | URL | 대상 |
|--------|-----|------|
| Professional | http://localhost:4040/pro | 의료진 |
| Consumer | http://localhost:4040/app | 환자 |

### 로그인 방법

#### 1. 간편 인증 (PIN)

```
1. 이름 입력
2. 전화번호 입력
3. 접속 PIN 입력 (6자리)
4. 로그인 버튼 클릭
```

#### 2. Google OAuth

```
1. "Google로 로그인" 버튼 클릭
2. Google 계정 선택
3. 권한 승인
4. 자동 로그인 완료
```

## 8.4 자주 묻는 질문 (FAQ)

### 일반

**Q: 비밀번호를 잊어버렸어요.**
> A: 로그인 화면에서 "비밀번호 찾기"를 클릭하거나, Google OAuth를 사용하세요.

**Q: 다른 기기에서도 접속할 수 있나요?**
> A: 네, 동일한 계정으로 여러 기기에서 접속 가능합니다.

### Professional

**Q: 진단 결과를 수정할 수 있나요?**
> A: 환자 상세 페이지에서 해당 진단을 선택하여 수정할 수 있습니다.

**Q: 환자에게 직접 결과를 전송할 수 있나요?**
> A: 처방 권고 화면에서 "환자에게 전송" 버튼을 클릭하세요.

### Consumer

**Q: 이전 진단 기록을 볼 수 있나요?**
> A: 내 진단 페이지에서 모든 과거 진단 기록을 확인할 수 있습니다.

**Q: 가족의 결과도 볼 수 있나요?**
> A: 각 계정은 개인별로 관리됩니다. 가족 결과는 해당 가족 계정으로 로그인해야 합니다.

## 8.5 문제 해결

### 로그인 오류

| 증상 | 해결 방법 |
|------|----------|
| "인증 실패" | PIN 번호 확인, 대소문자 구분 |
| "세션 만료" | 다시 로그인 |
| Google 로그인 실패 | 브라우저 팝업 허용 확인 |

### 페이지 오류

| 증상 | 해결 방법 |
|------|----------|
| 페이지 로딩 안됨 | 새로고침 (Ctrl+F5) |
| 데이터 표시 안됨 | 로그아웃 후 재로그인 |
| 저장 실패 | 네트워크 연결 확인 |
