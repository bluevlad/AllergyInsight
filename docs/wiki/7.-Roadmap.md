| 기능 | 설명 | 우선순위 |

## 7.3 기능 백로그

### 높은 우선순위

- [ ] **푸시 알림**: 진단 결과 알림, 관리 리마인더
- [ ] **PDF 내보내기**: 진단 결과 / 처방 권고 PDF 생성
- [ ] **모바일 반응형**: Consumer 앱 모바일 최적화
- [ ] **검색 기능 강화**: 환자 검색, 진단 필터링

### 중간 우선순위

- [ ] **데이터 시각화**: 차트, 그래프 개선
- [ ] **일괄 입력**: 여러 환자 동시 진단 입력
- [ ] **알림 설정**: 사용자별 알림 설정
- [ ] **테마 지원**: 다크 모드

### 낮은 우선순위

- [ ] **소셜 로그인**: 카카오, 네이버 추가
- [ ] **환자 커뮤니티**: 정보 공유 게시판
- [ ] **의료진 포럼**: 케이스 공유

## 7.6 버전 관리 정책

### 시맨틱 버저닝

```
MAJOR.MINOR.PATCH

예: 2.1.3
    │ │ └── 버그 수정, 패치
    │ └──── 기능 추가 (하위 호환)
    └────── 대규모 변경 (하위 비호환)
```

### 브랜치 정책

| 브랜치 | 용도 | 배포 환경 |
|--------|------|----------|
| `main` | 안정 버전 | - |
| `develop` | 개발 통합 | Test |
| `prod` | 운영 배포 | Production |
| `feature/*` | 기능 개발 | Local |
| `hotfix/*` | 긴급 수정 | Production |
